代码使用“ \` code \` ”包裹起来 。eg:`some code here`。

文章统一放在  `_post` 文件夹中。
文章中如果需要显示代码，代码块用 {% highlight ruby %} some codes here {% endhighlight %} 包裹起来。
效果如下：

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}
